## DSA in JAVA

A course on Data Structures and Algorithms using Java.

### Prerequisites
- Java 11 or higher
- Maven 3.6 or higher

### Building the Project

```bash
# Clone the repository
git clone <repo-url>
cd dsa-in-java

# Build with Maven
mvn clean install

# Run the main program
mvn exec:java -Dexec.mainClass="Main"
```

### Project Structure
```
dsa-in-java/
├── app/
│   └── src/          # Java source files
├── pom.xml           # Maven configuration
└── README.md
```

### Dependencies
- **SLF4J API 2.0.9** - Logging facade
- **SLF4J Simple 2.0.9** - Simple logging implementation
