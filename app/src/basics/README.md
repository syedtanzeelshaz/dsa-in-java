# Java Basics Learning Path

## Learning Order (Recommended)
Follow this order to build understanding progressively:

1. **[Variables & Data Types](./variables/)** - Start here!
   - Primitive types, reference types, variable scope

2. **[Operators](./operators/)**
   - Arithmetic, logical, relational, assignment

3. **[Control Flow](./controlFlow/)**
   - if-else, switch, loops (for, while, do-while)

4. **[Methods](./methods/)**
   - Function definition, parameters, recursion

5. **[Error Handling](./errorHandling/)**
   - Try-catch-finally, exceptions, custom exceptions

6. **[Collections](./collections/)**
   - Arrays, ArrayList, HashMap, HashSet

## How to Use This Package

Each topic has:
- **`*Example.java`** - Study this first to understand concepts
- **`*Exercise.java`** - Complete TODO sections for practice
- **`README.md`** - Quick reference and key concepts

### Study Method
1. Read the topic README.md for overview
2. Run the Example.java file in your IDE
3. Study the code and comments carefully
4. Complete the Exercise.java exercises
5. Test your code by running it

## Quick Reference

### Run Examples
Each Example class has a `main()` method:
```bash
javac basics/variables/VariablesExample.java
java basics.variables.VariablesExample
```

### Quick Topics Reference
| Topic | Key Concepts |
|-------|-------------|
| Variables | byte, short, int, long, float, double, char, boolean, String |
| Operators | +, -, *, /, %, &&, \|\|, !, ?:, ++, -- |
| Control Flow | if-else, switch, for, while, do-while, break, continue |
