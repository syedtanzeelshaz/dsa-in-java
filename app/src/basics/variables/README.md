# Variables & Data Types

## Key Concepts
- **Variable**: A named container for storing a value in memory
- **Data Type**: Defines what kind of value can be stored

## Primitive Data Types (8 types)
| Type | Size | Range | Example |
|------|------|-------|---------|
| byte | 1 byte | -128 to 127 | `byte x = 10;` |
| short | 2 bytes | -32,768 to 32,767 | `short x = 100;` |
| int | 4 bytes | -2^31 to 2^31-1 | `int x = 1000;` |
| long | 8 bytes | -2^63 to 2^63-1 | `long x = 100000L;` |
| float | 4 bytes | ~6-7 decimal places | `float x = 3.14f;` |
| double | 8 bytes | ~15-16 decimal places | `double x = 3.14;` |
| char | 2 bytes | 0 to 65,535 | `char x = 'A';` |
| boolean | 1 bit | true / false | `boolean x = true;` |

## Reference Data Types
- String, Arrays, Objects, etc.
- Stored in heap memory, reference in stack

## Important Rules
- Variable names: `camelCase`, start with letter
- Constants: `UPPER_CASE` with `final` keyword
- Type matters: `3.14` is double, `3.14f` is float
